This website!

I am especially proud of this site as it's something that I put lots of effort into - especially making it mobile friendly. Keeping it small and simple was one of my main priorities.

I had previously set up the hosting for this website the hard way, using an EC2 instance based in Singapore üá∏üá¨ which I manually configured with HTTPD, renewing the SSL certificate myself manually in order to enable HTTPS. Fun stuff! I leant a lot setting that up, however I have since changed the hosting solution to a more simple deployment with less maintainance required.

Site specs:

This site is hosted on Cloudflare, which is a CDN (Content Delivery Network) that enables fast load times by caching this sites data in various locations around the world. More about Cloudflare:

  • Highly available, the Cloudflare network can reach 95% of the worlds population within approximately 50 ms.
  • Encrypted over SSL by default with no configuration required, Cloudflare enables HTTPS without me needing to manage any certificates.
  • Completely 100% free! Cloudflare has a free tier that is perfect for simple static sites like this. The primary cost of running this page is the yearly ~$10 cost for the domain.

The site itself is written in pure HTML and CSS, and was designed to be light and minimal. If you want to take a look at the raw HTML yourself, the site is publicly available on my github here.
